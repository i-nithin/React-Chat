# React-Chat
A social web application that helps users communicate.<br/>
Deployed App : https://react-chat-a3e29.web.app/

## Technologies Used
React<br/>
Firebase<br/>
Tailwind.css<br/>

## How to run locally?
Clone the repo
```bash
https://github.com/i-nithin/React-Chat.git
```
Go to project directory
```bash
cd React-Chat
```
Install dependencies
```bash
npm install
```
Rut the App
```bash
npm start
```
